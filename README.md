# Andrew da Silva — Portfolio

A personal portfolio site showcasing UX, product design, and creative technology work.

## Tech Stack

- **React** + **Vite** — fast dev server and optimized builds
- **React Router** — client-side routing
- **CSS Modules** — scoped component styles

## Projects Featured

- **Magick ML** — Visual AI development platform (Co-Founder / CDO)
- **Project 89** — Immersive AI-driven narrative experience
- **Raw Magic Chocolate** — Story-driven wellness brand
- **Gamemaker Demo** — AI-accelerated solo game development

## Getting Started

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Deployment

Built for deployment on **Netlify**. The `dist/` directory is generated by `npm run build`.
